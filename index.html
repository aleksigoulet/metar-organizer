METAR Analyzer

Analyze your METAR data using the tool below.

The tool accepts .csv files containing historical data for a single location. Data entries must be of the format "station,valid,metar".

Files can be generated using ISU's tool available here.

Steps to download file using ISU's tool:

  1. Select station.
  2. Make sure to select "Raw METAR" in step 2 ("Select from Available Data").
  3. Specify date range.
  4. Make sure to select "UTC time" in step 4 ("Timezone of Observation Times").
  5. Select the following in step 5:
    • Date Format: "Comma Delimited"
    • Include Lat/long: "NO"
    • Include Elevation: "NO"
    • How to represent missing data: "Use 'M'"
    • How to represent Trace reports: "Use 'T'"
    • "Save result to computer"
  6. Upload file below.
